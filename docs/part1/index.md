# Part I — AI와 인문학의 만남

인공지능은 더 이상 공학자만의 영역이 아닙니다. 글을 쓰고, 번역하고, 이미지를 만들어내는 생성형 AI가 등장하면서 인문학도에게도 AI는 일상적인 도구가 되어가고 있습니다.

Part I에서는 두 가지 질문에 답합니다.

**1장**에서는 디지털 인문학의 흐름을 되짚으며, AI가 인문학의 어떤 영역을 바꾸고 있는지 살펴봅니다. 글쓰기와 창작, 텍스트 분석, 번역, 시각적 스토리텔링 — 이 네 영역에서 AI는 이미 새로운 가능성을 열어놓았습니다.

**2장**에서는 생성형 AI 자체를 이해합니다. 검색 AI와 생성형 AI의 차이, AI 기술의 계층 구조, 그리고 AI가 학습하는 데이터의 특성과 한계를 다룹니다. '데이터의 한계가 곧 AI의 한계'라는 통찰은 AI를 비판적으로 활용하기 위한 출발점이 됩니다.
