# 프롬프트 템플릿 모음

바로 복사해서 쓸 수 있는 프롬프트 템플릿입니다. `[대괄호]` 안의 내용을 자신의 상황에 맞게 바꿔 넣으세요.

## 글쓰기 템플릿

### 에세이 초안 작성

```
너는 [전공 분야] 전문 작가야.
[주제]에 대해 [대상 독자]를 위한 에세이를 써줘.

- 분량: [자수]자 내외
- 구성: 서론-본론(소제목 3개)-결론
- 문체: [격식/비격식] 산문체
- 구체적인 사례나 인용을 포함해줘
```

### 자기소개서

```
나는 [학과] [학년]이고, [경험/활동]을 해왔어.
[지원처]에 제출할 자기소개서를 써줘.

- 나의 강점: [강점 3가지]
- 강조하고 싶은 경험: [경험]
- 분량: [자수]자
- 진솔하고 구체적인 톤으로 써줘
```

## 분석 템플릿

### 문학 작품 분석

```
너는 [시대/장르] 전문 문학 비평가야.
[작품명]([작가])를 다음 관점에서 분석해줘:

1. [관점 1] (예: 주제 의식)
2. [관점 2] (예: 서사 구조)
3. [관점 3] (예: 인물 묘사)

각 관점별로 작품 속 구체적인 대목을 인용하며 설명해줘.
분량은 [자수]자 내외로.
```

### 비교 분석

```
다음 두 텍스트를 비교 분석해줘.
비교 기준: [기준 1], [기준 2], [기준 3], [기준 4]
결과를 표로 정리하고, 마지막에 종합 평가를 3줄로 써줘.

[텍스트 A 붙여넣기]
[텍스트 B 붙여넣기]
```

## 말투 변환 템플릿

### 범용 말투 변환 공식

```
다음 글을 [화자]가 [독자]에게 [매체/상황]에서
전달하는 말투로 다시 써 주세요.
- [종결 어미 스타일]
- [비유/예시 사용 여부]
- [전문 용어 처리 방식]
- [문장 길이/호흡]

[원문 붙여넣기]
```

### 구체적 말투 지정 예시

=== "강의형"
    ```
    다음 글을 "대학 강의에서 학생들에게 설명하는 교수"의
    말투로 다시 써 주세요.
    - 개념을 하나씩 풀어서 설명하세요
    - "~거든요", "~인데요" 같은 구어체 종결 어미를 사용하세요
    - 비유나 예시를 넣어 이해를 도와주세요
    ```

=== "보고서형"
    ```
    다음 글을 "기업 임원에게 보고하는 공식 기술 보고서"의
    말투로 다시 써 주세요.
    - 격식 있는 문어체를 사용하세요
    - 수치와 효과를 명확히 제시하세요
    - 감정적 표현을 배제하세요
    ```

=== "기사형"
    ```
    다음 글을 "IT 전문 매체 기자가 쓴 기술 리뷰 기사"의
    말투로 다시 써 주세요.
    - 제3자 시점에서 객관적으로 서술하세요
    - "~한 것으로 알려졌다" 같은 보도 문체를 사용하세요
    - 업계 동향과 연결 지어 서술하세요
    ```

## 평가·교정 템플릿

### AI 글 종합 평가

```
다음 글을 아래 6가지 기준으로 평가해 주세요.
각 항목마다 (O) 양호 또는 (X) 개선 필요를 표시하고,
(X)인 경우 해당 문장을 인용한 뒤 교정안을 제시해 주세요.

1. 문장 연결: 접속 어미 없이 짧게 끊긴 문장이 있는가?
2. 문단 분절: 한두 문장만으로 이루어진 문단이 있는가?
3. 번역투: "~하는 것이 중요합니다", "~에 있어" 등이 있는가?
4. 슬롭 워드: "혁신적인", "이를 통해" 등 AI 과용 단어가 있는가?
5. 종결 어미: "~합니다/~있습니다"만 반복되지 않는가?
6. 메타 해설: "핵심은~", "사례는~" 등 투어 가이드 문장이 있는가?

[평가할 글 붙여넣기]
```

### 교정 요청

```
다음 글에서 AI가 쓴 것 같은 부분을 모두 찾아 주세요.
각 부분에 대해:
1. 원문 인용
2. 어떤 AI 패턴에 해당하는지
3. 교정안
을 표로 정리해 주세요.

[교정할 글 붙여넣기]
```

## 학습 지원 템플릿

### 개념 설명 요청

```
[개념]을 [대상]이 이해할 수 있게 설명해줘.
- 비유를 1~2개 넣어줘
- 핵심만 3줄로 먼저 요약하고, 아래에 상세 설명을 붙여줘
- 관련 예시를 2개 들어줘
```

### 멀티턴 심화 전략

```
[첫 질문]  → 일반적인 답변을 받은 뒤

[후속 1]   "좀 더 구체적으로, [세부 주제]에 집중해줘."
[후속 2]   "그중에서 [특정 사례]를 예로 들어 설명해줘."
[후속 3]   "이 내용을 [형식]으로 정리해줘."
[후속 4]   "이 글의 문제점을 지적해줘."
```
