# Part II — AI와 대화하기

AI의 능력을 최대한 끌어내는 열쇠는 프롬프트에 있습니다. 같은 AI라도 어떻게 묻느냐에 따라 답변 품질이 크게 달라지는데, Part II에서는 프롬프트 작성의 기본 원리부터 AI가 쓴 글의 문제를 식별하고 교정하는 방법까지 다룹니다.

**3장**에서는 프롬프트의 개념과 4원칙(구체성, 맥락, 역할, 출력 형식)을 배웁니다. 모호한 질문과 구체적인 질문의 차이를 Before/After 예시로 확인하고, 멀티턴 대화 전략도 살펴봅니다.

**4장**에서는 같은 내용을 다른 말투로 바꾸는 실험을 합니다. 강의형, 경험자형, 보고서형, 선배형, 마케팅형, 기사형 — 6가지 말투의 실제 결과를 비교하며, 말투를 결정하는 네 가지 요소(화자, 독자, 종결 어미, 비유)의 조합을 배웁니다.

**5장**에서는 AI가 쓴 글의 전형적 문제 6가지를 분석합니다. 번역투, 슬롭 워드, 메타 해설 같은 패턴을 식별하는 눈을 기르고, AI 글을 자연스러운 한국어로 교정하는 기준을 세웁니다.
